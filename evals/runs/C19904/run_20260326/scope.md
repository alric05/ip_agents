# Invention Scope

## Customer Idea
A novel design of floating structure for eco-adaptive floating photovoltaic system: eco-float voltaic platform (EFVP). A modular floating PV system using a rectangular float with a divided top platform and four walls, an internal cross-shaped support forming four openings that retain four pontoons, triangular supports mounting a bifacial PV panel, mooring via concrete blocks and plastic-coated stainless steel wires to corners, and an integrated sensor/data-logger system wirelessly transmitting PV/environmental and electrical measurements to a central database.

## Confirmed Defaults (User confirmed)
1. Environment: inland reservoirs/lakes.
2. Float/pontoon material: HDPE rotational molded parts.
3. Pontoon retention: mechanical fastening/brackets.
4. Tilt: fixed tilt via triangular supports.
5. Wireless: cellular (LTE/NB-IoT).
6. Granularity: both single module and assembled multi-module array.

## Confirmed Scope
Novelty search will focus on (i) the specific modular float architecture (rectangular float + cross-shaped internal support retaining four separate pontoons), (ii) bifacial PV mounting using two triangular supports on divided platform, (iii) mooring/anchoring via concrete blocks + coated stainless wires to corners, and (iv) integrated monitoring (top/bottom PV temperature + irradiance + current/voltage, logger, wireless to central DB). Power export cable is considered conventional background unless claimed in a special way.
