## Submission Checklist

**Part A — Invention Disclosure:**
- [ ] Part A complete (all required fields filled in)
- [ ] Disclosure is 200–800 words
- [ ] Disclosure does NOT contain patent numbers, search strategies, or novelty conclusions
- [ ] Client information redacted (if from a real engagement)

**Part B — Ground Truth:**
- [ ] Section 1 complete (metadata + time breakdown)
- [ ] Section 2 complete (difficulty rating)
- [ ] Section 3 complete (3–7 features defined)
- [ ] Section 4 complete (all references listed; detail blocks for each A and B ref)
- [ ] Section 5 complete (all queries recorded with result counts)
- [ ] Section 6 complete (verdict + per-feature risk)
- [ ] Every A-ref has pin-cites
- [ ] Every A/B-ref has Y/Y1/N coverage for ALL features
